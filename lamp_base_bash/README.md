Base Lamp Stack
===============

This will be a base LAMP stack with current stable releases. For bleeding edge tech, use BE LAMP Stack instead.

--------

* ubuntu/trusty64
* Apache 2
* MySQL
* PHP 5.6
* PHPMyAdmin
* git
* composer
